# Day 8: Advanced Text Parsing

## Tools Practiced
- awk (field extraction, sorting)
- grep (log filtering)
- ps, sort, head
- wc

## Key Learnings
- awk -F helps split structured files
- ps aux + sort is powerful for resource analysis
- Logs can be parsed directly without script becoming complex
- set -euo pipefail improves script safety

## Real-world use
- SSH attack detection
- Resource monitoring
- Log analysis for troubleshooting
